Welcome to CandyLinux

The 🍭🍬🍫SWEETEST🍭🍬🍫 operating system!

CandyLinux is a lightweight Linux distribution designed by kids. For kids. (and beginners)

It is based on Ubuntu™ Server 24.04.2 LTS and features the lightweight Xfce desktop environment.

✨ Features of CandyLinux:

  • 🖥️ Lightweight and fast
  • 🎨 Customizable themes
  • 🔐 Secure and stable
  • 📦 Supports .deb and Snap packages
  • 🎮 Compatible with many Linux games
  • 💡 Perfect for beginners and tech lovers!
